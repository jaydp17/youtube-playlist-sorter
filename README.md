## :arrow_forward: Usage
Create an `API_KEY` ([how to create the key](https://developers.google.com/youtube/v3/getting-started)) and put it in your `~/.bashrc`.

```sh
echo 'export YOUTUBE_PLAYLIST_SORTER_API_KEY=AIzaxxxxxxxyyyyyyyyzzz' >> ~/.bashrc
source ~/.bashrc
```
